### 啟動注意事項
Git clone下來的檔案缺少db.properties，因為這個檔案是資料庫連線的私密資料，請找組長取得。

取得後的檔案請放到resources，剛抓下來可能會沒有這個資料夾，請自己創建一下。
```
專案資料夾/src/main/resources/db.properties
```

使用Java 1.8基本上不會有問題，更高版本不確定

* Java11經過測試可以啟動，但目前還是統一用1.8(除非老師後面有說要改)
